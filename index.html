Configuration Guide
1. Upload Word Document

Upload any Microsoft Word document (.docx). The application runs entirely client-side in your browser for total privacy and security.

2. Parent Wrapper Modes

Repeat Parent (Mode A): Wraps every section under a heading into a separate wrapper tag (e.g. <div class="blog-info-details">).
Single Parent (Mode B): Wraps the entire document into one single outer wrapper.

3. Tag & Class Rules

Configure output tags and CSS classes for Headings (H1-H6), Paragraphs, Lists (UL/OL/LI), Tables, Cells, and Images. Empty class/attribute fields will be omitted from the output automatically.

4. Hyperlink Manager

Define phrase replacement rules with custom URLs, targets, and optional Match Bold Only mode to target text inside bold Word formatting.
