How to Use the Template Converter
1. Uploading your document

Drag & Drop any Microsoft Word document (.docx) into the upload area. The system handles local reading in the browser. No files are uploaded to any server, making the process secure.

2. Defining HTML structure blocks

Instead of uploading one giant template, you define specific reusable HTML blocks in the builder. The converter automatically determines which template block to use by analyzing the structure of the document.

Detected Element Target Template Available Variables
Heading + subsequent text paragraphs Content Section {{heading}} (H2-H6), {{paragraph}} (HTML content of paragraphs)
Word Document Tables Table Section {{table}} (HTML structure of table rows/cells)
Bullet points / Numbered lists List Section {{list}} (Generated list items in <li> tags)
Embedded images Image Section {{image}} (Encoded base64 <img> element)
3. Variable substitution examples

Each block supports substitution tags. For instance, the default Content Block looks like this:

<section class="content">
    <div class="container">
        <h2>{{heading}}</h2>
        <p>{{paragraph}}</p>
    </div>
</section>
